Task Manager System Description

The Task Manager is a simple and user-friendly web-based application designed to help users organize, track, and manage their daily tasks efficiently. It allows users to create tasks, assign their status as either *Pending* or *Completed*, and monitor progress through a clean dashboard interface.

The system provides a centralized dashboard that displays real-time task statistics, including total tasks, completed tasks, and pending tasks. A visual chart enhances user understanding by representing task distribution in an easy-to-read format.

Users can add new tasks through a dedicated form, edit existing tasks, and view all tasks in a structured list. All task data is stored locally using browser storage, ensuring quick access without requiring a backend database.

Overall, the Task Manager improves productivity by helping users stay organized, prioritize work, and keep track of their progress in a simple and intuitive way.
